# Clasificador de Residuos con IA

Aplicación Streamlit para clasificar residuos reciclables usando un modelo MobileNetV2.

## Clases
- Cartón (cardboard)
- Vidrio (glass)
- Metal (metal)
- Papel (paper)
- Plástico (plastic)
- Basura (trash)

## Despliegue en Streamlit Cloud

1. Sube este repositorio a GitHub
2. Ve a [share.streamlit.io](https://share.streamlit.io)
3. Conecta tu repositorio de GitHub
4. Selecciona `app.py` como archivo principal
5. Haz clic en "Deploy"

## Archivos importantes
- `app.py` - Aplicación Streamlit
- `requirements.txt` - Dependencias
- `modelo_reciclaje_mobilenet/` - Modelo entrenado (agregar a .gitignore si es muy grande)
